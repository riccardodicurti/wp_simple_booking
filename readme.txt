## Description

Simple Booking for WordPress is a plugin that simplifies the integration of the Simple Booking bar and forms into your WordPress site.

### Features
* **Quick Setup:** Enter your Simple Booking ID directly on the plugin settings page.
* **WPML Compatibility:** If WPML is active, the plugin automatically uses the page's language, overriding the language setting in the options page.
* **ACF Support:** If ACF is active, you can set a different code or language for each page directly in the editor. For advanced configurations and detailed instructions, visit the plugin options page in your WordPress admin panel.
